# The Repair Intel

AI-Powered Home Inspection Cost Analysis Platform

## Features

- Upload PDF inspection reports
- AI-powered cost analysis (Handyman to Contractor pricing)
- Automated email delivery
- Stripe payment processing
- Customer database for marketing
- Pest alert system for Green Mantis Pest Control

## Tech Stack

- **Backend:** Node.js, Express
- **Database:** SQLite
- **AI:** Anthropic Claude API
- **Payment:** Stripe
- **Email:** Nodemailer (Gmail/Google Workspace)

## Environment Variables

Required environment variables (see `.env.example`):

- Stripe keys (payment processing)
- Anthropic API key (AI analysis)
- SMTP credentials (email delivery)
- Business settings

## Deployment

Deployed on Vercel at: www.TheRepairIntel.com

## Setup

1. Clone repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your keys
4. Run `npm start`
